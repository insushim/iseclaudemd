### 2. 학습 목표 작성 (Bloom's Taxonomy)

```yaml
인지적 영역:
  기억: "~을(를) 나열할 수 있다"
  이해: "~을(를) 설명할 수 있다"
  적용: "~을(를) 활용할 수 있다"
  분석: "~을(를) 비교/분석할 수 있다"
  평가: "~을(를) 평가할 수 있다"
  창조: "~을(를) 설계/개발할 수 있다"

예시:
  - "Python의 리스트 자료형을 설명할 수 있다" (이해)
  - "반복문을 활용하여 문제를 해결할 수 있다" (적용)
  - "두 알고리즘의 효율성을 비교할 수 있다" (분석)
```

### 4. 퀴즈 및 평가 문제

#### 객관식
```markdown
**문제 1.** Python에서 리스트의 마지막 요소에 접근하는 방법은?

A) list[0]
B) list[-1]
C) list[last]
D) list.end()

**정답:** B
**해설:** Python에서 음수 인덱스 -1은 마지막 요소를 가리킵니다.
```

#### 코딩 문제
```markdown
**문제.** 주어진 리스트에서 짝수만 필터링하는 함수를 작성하세요.

**입력 예시:**
```python
numbers = [1, 2, 3, 4, 5, 6]
```

**출력 예시:**
```python
[2, 4, 6]
```

**모범 답안:**
```python
def filter_even(numbers):
    return [n for n in numbers if n % 2 == 0]
```

**채점 기준:**
- 정확한 출력 (60%)
- 리스트 컴프리헨션 사용 (20%)
- 함수 정의 형식 (20%)
```
